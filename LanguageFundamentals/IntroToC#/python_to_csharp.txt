Print to console
Python:
print('thing we want to print to the console')

C#:
Console.WriteLine("thing we want to print to the console");

In C#, the double quote is reserved for strings, while the single quote
is reserved for chars

Variable Declaration:
Python:
x = 10
y = 'bologna'

C#:
int x = 10;
string word = "bologna";

Loops:
Python:
for i in range(len(arr)):
    print(arr[i])

C#:
for(int i = 0; i < Nums.Length; i++)
{
    Console.WriteLine(Nums[i]);
}

Conditionals:
Python:
if x == 10:
    // do thing here

C#:
if (x == 10)
{
    // do thing here
}
